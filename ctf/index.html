Mini CTF · 제한시간 30분

<보이지만 안 보이는 것들>

웹페이지는 당신에게 보여주는 것보다 훨씬 많은 걸 들고 있습니다. 여기 6개의 평범해 보이는 페이지가 있고, 각각 플래그를 하나씩 숨기고 있습니다. 아무것도 설치할 필요 없습니다. 브라우저 하나면 됩니다.

시작하기 전에 딱 하나

F12 를 누르세요. (맥은 ⌥⌘I) 개발자 도구가 열립니다. 이 30분 동안 쓸 도구는 이거 하나뿐입니다. 플래그는 전부 FLAG{...} 형태이고, 찾으면 중괄호까지 통째로 복사해서 제출하면 됩니다.

Challenges
▶01 <challenge title="개발자의 혼잣말" points="10">

동아리 신입 모집 페이지입니다. 아주 평범하죠. 그런데 개발자들은 코드에 자기들끼리 보는 메모를 남기는 습관이 있습니다.

배우는 것 — 페이지 소스 보기
페이지 열기
힌트 보기

페이지에서 마우스 오른쪽 클릭 → "페이지 소스 보기". 아니면 Ctrl+U. 브라우저가 보여주는 화면 말고, 브라우저가 받은 것을 보세요.

▶02 <challenge title="당첨자 명단" points="15">

경품 당첨자 명단이 붙었습니다. 그런데 명단이 좀 짧아 보이지 않나요? 안 보인다고 없는 건 아닙니다.

배우는 것 — CSS는 보안이 아니다
페이지 열기
힌트 보기

가장 게으른 방법부터 해보세요. Ctrl+A 로 페이지 전체를 드래그하면 어떻게 되나요? 글자가 사라진 게 아니라 색깔만 바뀐 거라면?

▶03 <challenge title="검색엔진은 오지 마세요" points="20">

웹사이트는 구글 크롤러에게 "여기는 들어오지 마"라고 부탁하는 파일을 둘 수 있습니다. 문제는, 그 부탁 목록이 공개라는 겁니다.

배우는 것 — 링크 없는 페이지도 존재한다
페이지 열기
힌트 보기

거의 모든 사이트의 최상위 주소 뒤에 /robots.txt 를 붙여보면 뭔가 나옵니다. 이 사이트도요.

▶04 <challenge title="나는 관리자다" points="30">

관리자만 볼 수 있는 페이지. 당신은 관리자가 아닙니다. 그런데 이 사이트는 당신이 관리자인지 아닌지를 당신에게 물어봅니다.

배우는 것 — 쿠키, 그리고 클라이언트를 믿으면 안 되는 이유
페이지 열기
힌트 보기

개발자 도구 → Application 탭 (파이어폭스는 저장소) → Cookies. 값이 하나 들어있습니다. 더블클릭하면 고칠 수 있고, 고친 다음 F5.

▶05 <challenge title="비밀번호를 아는 자" points="35">

비밀번호를 넣어야 플래그가 나옵니다. 비밀번호는 안 알려줍니다. 그런데 비밀번호를 검사하는 코드는 당신 컴퓨터에서 돌아갑니다.

배우는 것 — 자바스크립트 읽기, 콘솔 쓰기
페이지 열기
힌트 보기

페이지 소스에 <script> 가 있습니다. 검사하는 쪽 코드를 읽으면 정답도 거기 있습니다. 이상한 문자열이 나오면 Console 탭에 atob("...") 를 쳐보세요.

▶06 <challenge title="화면에 안 그려진 데이터" points="40">

공지사항 목록입니다. 서버에서 데이터를 받아와서 화면에 뿌립니다. 그런데 받아온 것과 뿌린 것이 같다는 보장이 있나요?

배우는 것 — Network 탭, 화면 뒤의 통신
페이지 열기
힌트 보기

개발자 도구 → Network 탭을 켜둔 채로 F5. 목록에 뜬 항목을 클릭해서 Response 를 보세요.

막히면 힌트를 여세요. 감점 없습니다. — 30분 안에 6개 다 못 풀어도 정상입니다.
